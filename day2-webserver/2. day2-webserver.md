1. Tujuan
- Menjalankan Apache lalu memvalidasinya
- Berpindah ke Nginx dan memastikannya bisa serve halaman
- Membaca log service via journalctl 
- Mengelola service via systemctl

2. Environment/Lingkungan
- Host: Endeavouros
- Guest: Ubuntu Server
- VM: QEMU
- Akses: ssh + tmux
- Tools: systemctl, journalctl, nano, ufw, web browser

3. Langkah Kerja
- Install apache
```bash
	sudo apt install apache2 -y
	sudo systemctl enable apache2
	sudo systemctl start apache2
	sudo systemctl status apache2
```

- Menghentikan Apache
```bash
sudo systemctl stop apache2
```

- Menginstall Nginx
```bash
sudo apt install nginx -y
sudo systemctl enable nginx
sudo systemctl start nginx
```

- Membuat halaman html sederhana untuk nginx
```bash
echo "<h1>nginx berjalan lancar!</h1>" | sudo tee /var/www/html/index.html
```

- Cek Log Nginx
```bash
journalctl -u nginx
```
